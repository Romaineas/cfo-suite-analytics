# Financial Health Strategic Dashboard 

Visão Geral

Este é um dashboard de Saúde Financeira Corporativa projetado para o alto escalão (C-Level). 

# Visualizar Dashboard Online 

KPIs Estratégicos Monitorados
Para garantir uma análise de saúde financeira robusta, o painel foca em quatro pilares:

Receita Líquida: O faturamento real após deduções e devoluções.

EBITDA: Lucro antes de juros, impostos, depreciação e amortização (essencial para medir a eficiência operacional).

Margem Líquida: A porcentagem de lucro real para cada real faturado.

Fluxo de Caixa: Monitoramento da liquidez imediata para garantir a solvência da operação.

# Visualizações e Insights

Performance Operacional (Mensal): Gráfico comparativo entre Receita e Despesas, permitindo identificar rapidamente meses de compressão de margem.

Distribuição de OPEX: Gráfico de rosca detalhando para onde está indo o capital da empresa (Folha, Marketing, Infraestrutura).

Break-even por Unidade de Negócio: Análise horizontal de lucratividade comparando diferentes canais (E-commerce, B2B, Lojas Físicas).

# Stack Tecnológica

Tailwind CSS: Para uma interface limpa, corporativa e totalmente responsiva.

Chart.js: Utilizado para renderizar gráficos de alta densidade de dados com interatividade.

JavaScript (Real-Time Logic): Simulação de oscilação de fluxo de caixa em tempo real para monitoramento contínuo.


# Este projeto simula uma estrutura de dados que seria alimentada por um pipeline robusto:

Ingestão: Dados extraídos de ERPs financeiros via SQL.

Processamento: Limpeza e padronização de contas contábeis utilizando PySpark/Databricks (especialmente para grandes volumes de transações).

Visualização: Entrega final focada na experiência do usuário (UX), removendo ruídos e destacando o que realmente impacta o lucro líquido.
